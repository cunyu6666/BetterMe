# Design DNA: BetterMe
> 生成日期: 2026-06-08 | 版本: 1.0

## 受众画像 (Audience)

- **主要用户**: 关注自身健康与成长的中国都市青年
- **年龄范围**: 22-35
- **技术素养**: 中高
- **文化背景**: 中国都市，偏好中文界面
- **使用场景**: 通勤手机浏览、睡前回顾、桌面专注记录
- **核心需求**: 习惯养成、健康追踪、目标管理、自我提升

## 品牌性格 (Brand Personality)

| 维度 | 分数 | 偏向 |
|------|------|------|
| 大胆 ←→ 保守 | 3 | 中性，不张扬 |
| 温暖 ←→ 专业 | 5 | 偏温暖 |
| 活泼 ←→ 沉稳 | 5 | 偏沉稳 |
| 奢华 ←→ 亲民 | 6 | 偏亲民 |
| 传统 ←→ 前卫 | 4 | 略偏前卫 |

**性格关键词**: 安静、温暖、专注、呼吸感、禅意

## 美学方向 (Aesthetic Direction)

- **行业定位**: Healthcare (#2) + Fitness (#11) 融合
- **主风格**: 简约禅意 (Minimal Zen)
- **副风格**: 温暖有机 (Warm Organic)
- **视觉参考**: "无印良品的留白 + 暖色点缀的生命力"

## 配色哲学 (Color Philosophy)

- **色彩空间**: OKLCH
- **主色**: `oklch(0.62 0.16 45)` — 暖珊瑚色，用于 CTA 和强调
- **辅色**: `oklch(0.72 0.12 35)` — 浅桃色，用于 hover 和次要元素
- **强调色**: `oklch(0.55 0.14 25)` — 深陶土色，用于 active 状态
- **中性灰基调**: 微暖色调 (hue 60-80)
- **情绪**: 安静沉稳的底色 + 温暖有生命力的点缀
- **暗色模式**: 是，独立设计（非反色）

## 排版基调 (Typography)

- **标题字体**: Noto Sans SC — 中文无衬线，清晰稳重
- **正文字体**: Noto Sans SC — 统一字体，用字重制造层次
- **字阶比率**: 1.25 (Major Third)
- **标题字重**: 700
- **正文字重**: 400
- **行宽偏好**: 65ch

## 间距风格 (Spacing)

- **基础单位**: 4pt
- **整体风格**: 宽松呼吸感
- **节奏特征**: 集群紧密，section 分离慷慨

## 动效态度 (Motion)

- **动效程度**: 极简
- **默认缓动**: ease-out
- **品牌动效**: 柔和渐显，无弹跳，无 bounce

## 可访问性 (Accessibility)

- **目标等级**: WCAG AA
- **特别注意**: 移动端触摸目标 ≥ 44px

## 技术栈 (Tech Stack)

- **框架**: React 19 + TypeScript + Vite 8
- **样式**: Tailwind CSS 4 + CSS 自定义属性
- **图标**: Tabler Icons (双色风格: line + partial fill)
- **字体**: Noto Sans SC (Google Fonts)

## 设计规则 (Design Rules)

1. 背景永远不是纯白 #fff，使用 `#faf9f7` 暖白
2. 文字永远不是纯黑 #000，使用 `#2c2825` 暖深棕
3. 卡片圆角 16px，内部间距 20px，卡片间距 16px
4. 主色仅用于 CTA 按钮和关键状态，不超过页面面积 10%
5. 中性灰带微暖色调，避免冷灰
6. 图标统一使用 Tabler，尺寸 20px（默认）/ 16px（紧凑）
7. 暗色模式背景使用 `#1e1c1a` 暖深色，非冷蓝黑
8. 所有动画尊重 prefers-reduced-motion
